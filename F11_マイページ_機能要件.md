# F11 マイページ 機能要件

## 1. 目的
- ログインユーザーにプロフィール情報とお気に入り情報を一元管理できる画面を提供し、継続利用と満足度向上を図る。

## 2. 対象範囲・前提
- 年齢確認（F01）通過済みかつログイン済み（F05）であること。
- 対象機能: プロフィール表示／ユーザー名変更、お気に入り動画／ジャンル／女優一覧の表示。

## 3. 機能概要
- マイページ画面でユーザー名などのプロフィール情報を表示し、編集画面へ遷移可能にする。
- お気に入り動画／ジャンル／女優一覧をタブまたはセクション単位で表示する。

## 4. 詳細要件
- アクセス制御
  - マイページ画面はログイン必須。未ログイン時はログイン画面へ誘導する。
- プロフィール表示
  - ユーザー名など基本情報を表示する。
  - ユーザー名編集（F14）画面への遷移リンクを設置する。
- お気に入り一覧表示
  - お気に入り動画／ジャンル／女優をタブやセクションで切り替えて表示する。
  - お気に入り動画は縦スクロール UI を利用し、動画視聴画面へ遷移可能とする。

## 5. 入出力
- 入力: ログイン済みユーザーのリクエスト。
- 出力: プロフィール情報、お気に入り動画／ジャンル／女優の一覧データ。

## 6. 関連
- F05 ユーザー管理: 認証・認可。
- F06/F12/F13 お気に入り機能: 一覧表示の元データ。
- F14 ユーザー名変更: プロフィール編集の詳細機能。
