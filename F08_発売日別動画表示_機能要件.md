# F08 発売日別動画表示 機能要件

## 1. 目的
- 発売日を切り口に、今日発売／明日発売／近日発売の動画を素早く一覧できるようにする。

## 2. 対象範囲・前提
- 年齢確認（F01）通過済みであること。
- ログイン／未ログインいずれでも利用できる。

## 3. 機能概要
- 「今日発売」「明日発売」「近日発売（期間は設計で定義）」といったプリセット検索を提供し、縦スクロール UI で結果を表示する。
- ページネーションに対応し、結果が尽きた際には終了メッセージを表示する。

## 4. 詳細要件
- プリセット条件
  - 今日発売: 当日発売日の作品を対象。
  - 明日発売: 翌日発売日の作品を対象。
  - 近日発売: 今週／来週など、具体的な期間は設計で定義。
- 実行
  - 選択されたプリセットに応じて検索 API を呼び出し、発売日条件を付与して結果を取得する。
  - ページネーション対応とし、ページ末尾で次ページ有無を確認する。
- 表示
  - タイトル、出演者、発売日、外部遷移リンクを各カードに表示する。
  - 結果が尽きた場合は「該当する動画がなくなりました」と表示する。
- ロギング
  - 選択されたプリセット種別、結果件数、エラー有無をログとして保存する。

## 5. 入出力
- 入力: プリセット種別（今日／明日／近日）、ページ番号。
- 出力: 発売日条件に合致する動画リスト（メタ情報、サンプル動画 URL、ページネーション情報）、イベントログ。

## 6. 関連
- F02 縦スクロール視聴・レコメンド: 発売日別結果表示に縦スクロール UI を利用。
- F03 動画メタ情報取得: メタ情報の取得に利用。
- F04 外部サービス遷移: 発売日別結果からの外部遷移。
